📖 Manual completo de la plataforma

Esta guía explica paso a paso cómo crear encuestas, distribuirlas y analizar los resultados. Todo se almacena en la base de datos de producción en tiempo real.

🚀 Sin instalación ☁️ 100% en la nube 📊 Resultados en vivo 📥 Exportación CSV
🔐

1. Acceso al panel de administración

Cómo ingresar al panel para gestionar todo

  1. 1
    Abre el panel admin

    Navega a la URL del panel:

    https://encuesta-tecnologia.onrender.com/admin/

    Si estás en local: http://localhost:8000/admin/

  2. 2
    Ingresa la clave de administrador

    Aparecerá una ventana de acceso. Escribe la ADMIN_KEY configurada en el servidor y presiona Enter o el botón Entrar.

    💡 La clave se guarda en la sesión del navegador. No necesitas volver a ingresarla mientras la pestaña esté abierta.
  3. 3
    Panel principal cargado

    Verás la lista de todas las encuestas creadas. Si es la primera vez, el panel estará vacío.

    ✅ Si ves la tabla de encuestas, el acceso fue exitoso.
✏️

2. Crear una nueva encuesta

Diseña tu encuesta con título, descripción y preguntas

  1. 1
    Haz clic en "+ Nueva encuesta"

    Botón rojo en la esquina superior derecha del panel principal. Abrirá el formulario de creación.

  2. 2
    Completa los datos generales

    Título — Nombre visible de la encuesta (obligatorio). Se usará para generar el slug/URL automáticamente.
    Descripción — Texto introductorio que verán los encuestados (opcional pero recomendado).

    💡 El título "Satisfacción del Cliente 2026" generará el link /encuesta.html?s=satisfaccion-del-cliente-2026
  3. 3
    Agrega preguntas

    Haz clic en "+ Agregar pregunta" para añadir cada pregunta. Para cada una debes definir:

    • Texto — Lo que le preguntas al encuestado
    • Tipo de respuesta — Ver sección 3
    • Opciones — Para radio, checkbox, select (una por línea)
    • Obligatoria — Marca el checkbox si la pregunta no puede quedar sin respuesta
  4. 4
    Ordena las preguntas

    Usa los botones ▲ ▼ en cada tarjeta de pregunta para cambiar el orden en que aparecerán.

  5. 5
    Guarda la encuesta

    Haz clic en "Guardar encuesta". El sistema la creará en la base de datos y te redirigirá al panel principal donde aparecerá como Activa.

    ✅ La encuesta se activa automáticamente al crearla y ya puede recibir respuestas.
🗂️

3. Tipos de pregunta disponibles

Elige el tipo adecuado según la respuesta que necesitas capturar

radio Opción única

El encuestado elige una sola opción de una lista de botones de radio.

Ej: ¿Cuál es tu nivel de satisfacción? → Bajo / Medio / Alto
checkbox Opción múltiple

El encuestado puede seleccionar varias opciones al mismo tiempo.

Ej: ¿Qué servicios usas? → Email / Chat / Teléfono / Portal
select Lista desplegable

Igual que radio pero en formato menú compacto. Ideal para listas largas (más de 5 opciones).

Ej: ¿En qué departamento trabajas? → [lista de áreas]
scale Escala numérica

Botones de selección en un rango numérico configurable (ej. 1 a 5 o 1 a 10). Incluye etiquetas en extremos.

Ej: Del 1 al 5, ¿qué tan probable es que recomiendes el servicio? (NPS)
text Texto corto

Campo de una sola línea para respuestas breves como nombre, correo, cargo, etc.

Ej: ¿Cuál es tu nombre completo?
textarea Texto largo

Área de texto multilínea para comentarios, sugerencias o respuestas extensas.

Ej: ¿Tienes alguna sugerencia para mejorar el servicio?
ℹ️ Para los tipos radio, checkbox y select, debes escribir cada opción en una línea separada en el campo "Opciones". Para scale, configura el valor mínimo, máximo y sus etiquetas.
🔗

4. Compartir el link de la encuesta

Cómo obtener y distribuir el enlace único de cada encuesta

  1. 1
    Ubica la encuesta en el panel principal

    En la tabla verás la columna "Slug / Link" con el identificador único de cada encuesta.

  2. 2
    Copia el link con el botón 📋

    Haz clic en "📋 Copiar link". El enlace completo se copia al portapapeles automáticamente. Tendrá este formato:

    https://encuesta-tecnologia.onrender.com/encuesta.html?s=nombre-de-tu-encuesta
  3. 3
    Distribuye el link

    Comparte el enlace por cualquier canal: correo electrónico, WhatsApp, Slack, Teams, QR, redes sociales, etc. Cualquier persona con el link puede responder sin necesidad de cuenta.

    💡 La encuesta debe estar en estado Activa para recibir respuestas. Si está Pausada, el encuestado verá un mensaje de no disponible.
👤

5. Experiencia del encuestado

Lo que ve y hace la persona que recibe el link

  1. 1
    Abre el link recibido

    El encuestado abre el link en cualquier navegador (móvil o escritorio). No requiere cuenta ni instalación.

  2. 2
    Ve el título y descripción

    La página muestra el título, descripción y el número total de preguntas.

  3. 3
    Responde las preguntas

    Completa todas las preguntas marcadas como obligatorias. Las preguntas opcionales pueden dejarse en blanco.

  4. 4
    Envía la respuesta

    Al presionar "Enviar respuesta", el formulario se valida. Si falta alguna pregunta obligatoria, se resalta en rojo. Si todo está correcto, la respuesta se guarda en la base de datos.

    ✅ El encuestado ve una pantalla de confirmación. La respuesta queda registrada de forma permanente.
⚠️ Actualmente la plataforma permite responder múltiples veces desde el mismo dispositivo. Si necesitas respuestas únicas por persona, considera distribuir el link de forma controlada (correo nominativo, etc.).
📊

6. Ver resultados en tiempo real

Analiza las respuestas con gráficas automáticas

  1. 1
    Abre los resultados desde el panel

    En la tabla de encuestas haz clic en el botón "📊 Resultados" de la encuesta que quieras analizar.

  2. 2
    Resumen general

    En la parte superior verás las métricas globales:

📝
Total de respuestas

Número de personas que completaron la encuesta

❓
Preguntas

Cantidad de preguntas en la encuesta

🕐
Última respuesta

Fecha y hora del registro más reciente

  1. 3
    Gráficas por pregunta

    Cada pregunta muestra sus resultados de forma visual:

    • radio / select / scale → Gráfica de dona con porcentajes
    • checkbox → Gráfica de barras horizontales
    • text / textarea → Lista de respuestas de texto con conteo de apariciones
    💡 Las gráficas se actualizan en tiempo real cada vez que entras a la página de resultados.
📥

7. Exportar resultados a CSV

Descarga todos los datos para análisis en Excel o Google Sheets

  1. 1
    Desde la página de resultados

    Haz clic en el botón "📥 Exportar CSV" en la barra superior de resultados.

  2. 2
    Archivo descargado

    Se descargará un archivo resultados-[slug].csv con la siguiente estructura:

    # Columnas del CSV response_id | submitted_at | Pregunta 1 | Pregunta 2 | ... 1 | 2026-04-15 | Muy satisfecho | Sí | ... 2 | 2026-04-15 | Neutral | No | ...
  3. 3
    Abre en Excel o Google Sheets

    El CSV usa codificación UTF-8 y separador de coma. En Excel: Datos → Desde texto/CSV. En Google Sheets: Archivo → Importar.

    ✅ Desde el CSV puedes crear tablas dinámicas, filtros, gráficas personalizadas y análisis avanzado.
⚙️

8. Gestionar encuestas existentes

Pausar, activar y eliminar encuestas desde el panel

Acción Botón Descripción Efecto inmediato
Pausar ⏸ Pausar Desactiva temporalmente la encuesta El link muestra "No disponible" a nuevos encuestados. Los datos existentes se conservan.
Activar ▶ Activar Reactiva una encuesta pausada El link vuelve a funcionar y acepta respuestas de nuevo.
Ver resultados 📊 Resultados Abre la página de análisis Muestra gráficas y datos en tiempo real.
Eliminar 🗑 Eliminar Borra permanentemente la encuesta Se eliminan la encuesta, todas sus preguntas y todas las respuestas. No se puede deshacer.
⚠️ Importante: La eliminación es permanente. Si solo quieres cerrar la encuesta temporalmente, usa Pausar. Elimina solo si ya no necesitas ninguno de los datos.
🔌

9. API REST / Integración avanzada

Crea encuestas programáticamente y consulta resultados desde código

  • ✓ La documentación interactiva de la API está disponible en /docs (Swagger UI)
  • ✓ Todas las rutas admin requieren el header X-Admin-Key: tu_clave
  • ✓ Las rutas públicas (responder encuesta) no requieren autenticación

Ejemplo: crear encuesta via API (curl)

curl -X POST "https://encuesta-tecnologia.onrender.com/api/admin/surveys" \ -H "Content-Type: application/json" \ -H "X-Admin-Key: tu_clave" \ -d '{ "title": "Encuesta de Satisfacción", "description": "Cuéntanos tu experiencia", "questions": [ { "type": "radio", "label": "¿Cómo calificarías el servicio?", "options": ["Excelente", "Bueno", "Regular", "Malo"], "required": true, "order": 1 }, { "type": "textarea", "label": "¿Tienes algún comentario adicional?", "required": false, "order": 2 } ] }'

Endpoints disponibles

MétodoRutaDescripción

POST/api/admin/surveysCrear nueva encuesta

GET/api/admin/surveys/{slug}/exportExportar resultados CSV

DELETE/api/admin/surveys/{slug}Eliminar encuesta
